# Pontiletra

Gerador de folhas de caligrafia para educadores brasileiros, feito com Next.js
App Router.

## Como rodar

```bash
npm install
npm run dev
```

A aplicação fica disponível em `http://localhost:3000/pontiletra`. A rota `/`
redireciona automaticamente para o gerador.

## Scripts

```bash
npm run build
npm run lint
npm run start
```
